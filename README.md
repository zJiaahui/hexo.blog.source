## 下载克隆到本地
```bash
git clone https://github.com/zJiaahui/hexo.blog.source.git
```
## 安装依赖
```bash
npm install
```
## 本地运行预览项目
```bash
hexo clean
```
```bash
hexo s
```
## 根据创建的本地服务进行预览

```
http://localhost:4000/hexo.blog/
```
